# 🍳 免開伙大廚
> 「大熱天不用顧爐火，夏天也能好好吃飯。記下每一次加熱，找出最適合自己的料理公式。」

這是一個專門為「微波爐與電鍋料理」設計的個人手帳式料理實驗紀錄資料庫。幫助用戶精準記錄每次加熱功率、水量、時間與成果，告別烹飪黑暗料理，複製成功美味！

## ✨ 核心特色
- **夏日清新風格**：採用溫暖奶油白、鼠尾草綠與淺黃色，打造高質感手帳筆記視覺。
- **雙模式架構**：未登入時全自動採用 `LocalStorage` 本地快取儲存；透過 Email Magic Link 登入後即開啟 `Supabase` 全球跨裝置即時同步，並支援「一鍵上雲端」無損搬移。
- **動態加熱控制**：針對微波爐（瓦數、加蓋、分段）與電鍋（外鍋水量、燜煮時間）進行模組化連動紀錄。
- **數據化分析**：首頁自動計算最常使用設備、隨機推薦歷史大成功的料理配方。

## 🛠️ 開發與環境配置
本網站為純前端 SPA 應用，可無縫部署於 GitHub Pages。

### 開啟雲端同步功能：
1. 註冊並登入 [Supabase 控制台](https://supabase.com/)。
2. 建立新專案，至 **SQL Editor** 貼入本專案提供的 SQL 建表指令完成初始化。
3. 前往專案的 **Project Settings -> API**，複製您的 `Project URL` 與 `anon public key`。
4. 在部署或本地執行時，將金鑰填入即可啟用跨裝置同步。
